## 📋 Description

This project is a functional keylogger that records and logs keystrokes for legitimate security research, parental control (on your own devices), or authorized penetration testing.

## ✨ Features

- Records all keyboard input
- Timestamps each keystroke
- Logs special keys (Enter, Space, Backspace, etc.)
- Saves logs to encrypted local file
- Optional email reporting functionality
- Runs in background (stealth mode optional - see config)
- Cross-platform support (Windows/Linux/macOS)
